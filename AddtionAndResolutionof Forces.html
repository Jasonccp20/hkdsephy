數學原理解析

Trigonometry & Vectors

1. 力的合成 (求合力 F)

利用畢氏定理 (Pythagoras Theorem) 尋找斜邊長度：

F² = Fx² + Fy²
F = √(40.0² + 30.0²)
F = 50.0 N

利用正切函數 (Tangent) 尋找角度：

tan(θ) = Fy / Fx
tan(θ) = 30.0 / 40.0
θ = 36.9°

2. 力的分解 (拆力 Fx, Fy)

給定合力 F 與角度 θ，利用三角比尋找分力：

尋找鄰邊 (Adjacent) - 水平分力：
cos(θ) = Fx / F
Fx = F × cos(θ)
Fx = 50.0 × cos(36.9°)
Fx = 40.0 N
尋找對邊 (Opposite) - 垂直分力：
sin(θ) = Fy / F
Fy = F × sin(θ)
Fy = 50.0 × sin(36.9°)
Fy = 30.0 N
